# PyMaze

一个用 PySide6 写的迷宫生成、求解的 GUI 程序。

## 功能特性

- 支持多种迷宫生成算法：
  - DFS (递归回塑)
  - Prim 算法
  - Kruskal 算法
  - Wilson 算法
- 支持多种寻路算法：
  - DFS 寻路
  - A-Star (A*) 寻路（支持最短路径）
- 支持调整迷宫尺寸（长宽可不等）。
- 支持随机拆墙功能（破坏完美迷宫，产生多条路径），用于演示 A* 算法的优势。
- 迷宫单元格自动居中且保持正方形。

## 安装与运行

1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

2. 运行程序：
   ```bash
   python maze.py
   ```

## 预览

启动程序后，你可以通过控制面板调整迷宫参数，点击“生成迷宫”即可创建新迷宫。点击“显示/隐藏路径”查看寻路效果。
